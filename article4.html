Всемирная паутина (WWW)
Схема перекрёстных ссылок в гипертекстовых документах, которую создал Бернерс-Ли.
URI/URL
Унифицированный идентификатор ресурса, который разработал Бернерс-Ли.
HTTP
Протокол передачи гипертекстовых документов, который создал Бернерс-Ли.
HTML
Язык гипертекстовой разметки, на котором создавал документы Бернерс-Ли.
Браузер
Специальная программа, с помощью которой можно получить доступ к документу по протоколу HTTP
Консорциум всемирной паутины (WЗС)
Организация, которую основал и возглавил Бернерс-Ли, разрабатывающая и контролирующая стандарты www.
Концепция гипертекста
Система, которая позволяет тексту ссылаться на другую информацию, реализованная Бернерс-Ли.
